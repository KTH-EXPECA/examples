# README

## `setup_sdr.ipynb`
This notebook is used to prepare and validate the SDR environment.

- Configure Mango/NI mode
- Reboot the SDR
- Perform a basic SDR functionality test

## `sdr_2pub_containers.ipynb`
This notebook is used to setup one SDR with two containers.

- 2 containers connected to 1 SDR
- 1 container assigned to RJ45
- 1 container assigned to SFP
- Both containers have public IP connection

## `2sdr_2containers.ipynb`
This notebook is used to setup 2 SDRs with one container each, and can be used for SDR-SDR testing

- 1 container per SDR, connected to the SFP interface
- Both containers have public IP connection
- Both containers are prepared with OAI
